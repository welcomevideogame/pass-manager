# pass-manager

This application is a lightweight password manager that allows for the storage and management of passwords in a secure and convenient way. It uses CustomTkinter for a more modern GUI and symmetric encryption to protect the passwords from unauthorized access.

## Features
- Add, edit, and delete passwords
- Filter for passwords by website
- Generate strong and unique passwords
- Encrypt and decrypt passwords
- Securely store passwords
- Protect the password manager with a key
- Option for fully offline password manager

## Technologies and frameworks
- Python
- CustomTkinter
- Truly random password generation

## Installation and usage
1. Install Python and a Python IDE such as PyCharm or IDLE.
2. Clone or download the repository
3. Install dependencies: `pip install -r requirements.txt`
4. Run the `main.py` script to start the application

## Target audience
This application is intended for individuals who want to store and manage their passwords in a secure and convenient way. It is designed to be easy to use and customize, with a focus on providing strong and reliable password protection. It is open source, providing a trustworthy password manager only you have access to.
